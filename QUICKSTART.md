# Быстрый старт

## Запуск проекта

1. Откройте терминал в корневой директории проекта

2. Запустите Docker контейнеры:
```bash
docker-compose up --build
```

3. Дождитесь сообщения "Database is ready!" и создания суперпользователя

4. Откройте в браузере:
   - **Галерея**: http://localhost:8000
   - **Админка**: http://localhost:8000/admin
     - Логин: `admin`
     - Пароль: `admin`

## Использование админки

1. Войдите в админку
2. Перейдите в раздел **Images** (Изображения)
3. На странице управления:
   - Выберите файл изображения
   - Введите имя
   - Нажмите **Загрузить**
   - Измените имя в поле рядом с изображением (сохраняется автоматически)
   - Нажмите **Удалить** для удаления изображения

## Остановка

Нажмите `Ctrl+C` в терминале или выполните:
```bash
docker-compose down
```

## Удаление всех данных

```bash
docker-compose down -v
```
